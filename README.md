E-Commerce Backend System (Spring Boot)
 --Project Overview

This project is a backend system for an E-commerce application built using Spring Boot. It provides RESTful APIs for managing users, products, carts, and orders.

The application follows a layered architecture (Controller → Service → Repository) and is designed for scalability and maintainability.

-- Features
 Authentication
User Registration & Login
Session-based authentication
Secure access to application features
--User Management
Create user accounts
Login and logout functionality
View user details
-- Product Management
Add new products
Update product details
Delete products
View all available products
--Cart Management
Add products to cart
Update quantity
Remove items
View total cart value
--Order Management
Place orders
View order history
🛠️ Tech Stack
Backend: Java, Spring Boot
Database: MySQL
Tools: Maven, Git
